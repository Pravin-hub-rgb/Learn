# 🧠 React Mastery Checklist

## 🟢 1. Core Basics (Must Clear)

- [ ] JSX kya hai aur kaise kaam karta hai  
- [ ] Functional components banana aata hai  
- [ ] Ek component ko dusre me import/use kar sakta hoon  
- [ ] Props pass karna aur receive karna aata hai  
- [ ] Destructuring props samajhta hoon  

---

## 🟢 2. State & Interactivity

- [ ] `useState` use kar sakta hoon  
- [ ] State update karte waqt spread operator use karna aata hai  
- [ ] Input field ko state se connect kar sakta hoon  
- [ ] Button click pe state update kar sakta hoon  
- [ ] Array state me add/delete/update kar sakta hoon  

---

## 🟢 3. Rendering Logic

- [ ] Conditional rendering (`&&`, ternary) use kar sakta hoon  
- [ ] `.map()` se list render kar sakta hoon  
- [ ] Proper `key` kyun important hai samajhta hoon  

---

## 🟢 4. Side Effects

- [ ] `useEffect` kab use karte hain samajhta hoon  
- [ ] Dependency array ka role clear hai  
- [ ] Component mount vs update ka difference samajhta hoon  

---

## 🟢 5. Forms Handling

- [ ] Controlled components kya hote hain pata hai  
- [ ] Form submit handle kar sakta hoon  
- [ ] Prevent default ka use pata hai  

---

## 🟢 6. Mini Project (Todo App)

Project me ye sab hona chahiye:

- [ ] Add task  
- [ ] Delete task  
- [ ] Mark complete  
- [ ] Filter (All / Completed / Pending)  
- [ ] LocalStorage me save ho  
- [ ] Alag-alag components properly split ho  

---

## 🟢 7. Confidence Test 🔥

Agar tu:
- [ ] Blank page se Todo bana sakta hai bina tutorial ke  
- [ ] Error aaye toh khud debug kar sakta hai  
- [ ] Kisi ko explain kar sakta hai ki state ka flow kaise chal raha hai
