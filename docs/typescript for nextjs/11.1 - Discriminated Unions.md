## 🚀 SABSE PEHLE PROBLEM DIKHTE HAI

Abhi tumhare paas ye action type hai:
```tsx
type ThemeAction = 
  | { type: "TOGGLE_THEME" }
  | { type: "SET_FONT_SIZE"; payload: number }
```

❌ **PROBLEM KYA HAI?**
Ab agar tum `TOGGLE_THEME` bhejte ho toh tum payload bhi bhej sakte ho.
Aur agar tum `SET_FONT_SIZE` bhejte ho toh tum payload nahi bhi bhej sakte ho.

TypeScript ko nahi pata hai ki konsa action mein payload chahiye aur konsa mein nahi.

## 🎯 Kya Problem Aayegi?

```tsx
// Ye chalega lekin galat hai

```

✅ TOGGLE_THEME mein payload nahi chahiye fir bhi ye chal raha hai.

```tsx
// Ye bhi chalega lekin galat hai

```

✅ SET_FONT_SIZE mein payload chahiye fir bhi ye chal raha hai.

Yeh TypeScript ki sabse badi kami hai is case mein.

❌ **Ab ye nahi chalega:**
```tsx

// ❌ ERROR: payload is not allowed here
```

```tsx

```

✅ **Ab sirf ye hi chalega:**
