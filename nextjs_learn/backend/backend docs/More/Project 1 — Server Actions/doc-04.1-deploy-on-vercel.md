## Deploy on Vercel Kya Hota Hai? (Real Life Analogy)

Deploy on Vercel ek **shop opening** jaisa hai:

| Shop Opening | App Deployment |
|--------------|----------------|
| Shop Location | Vercel Platform |
| Shop Setup | Project Configuration |
| Opening Ceremony | Deployment |
| Customers | Users |

Agar shop nahi khulta, kaise pata chalega ki kaun customer kaun aayega? Isliye shop khulna zaroori hai.

## Hum Kya Sikhenge?

Hum Deploy on Vercel sikhenge jisse:

1. **App live ho** - App publicly available ho
2. **Database connect ho** - Database securely connect ho
3. **Performance optimize ho** - App fast aur reliable ho
