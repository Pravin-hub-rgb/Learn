## Complex App Kya Hota Hai? (Real Life Analogy)

Complex App ek **shopping mall** jaisa hai:

| Shopping Mall | App Development |
|---------------|-----------------|
| Multiple Stores | Multiple Features |
| Different Sections | Different Modules |
| Central Management | Main App Logic |
| Customer Flow | User Experience |

Agar shopping mall nahi hota, kaise pata chalega ki kaun customer kaun kahan ja raha hai? Isliye mall zaroori hai.
