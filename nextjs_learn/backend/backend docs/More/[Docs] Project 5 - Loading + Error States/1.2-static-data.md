# Doc 1.2 — Movies Page — Static Data Se Dikhao

## Abhi Tak Kya Hua
1.1 mein Next.js app banaya, `/movies` route banaya, Home se link daala. App chal raha hai.

---

## Humara Goal Abhi

Movies page pe kuch movies dikhani hain. Par abhi koi API nahi, koi fetch nahi.

Kyun? Kyunki pehle UI samjhenge — data kaisa dikhega — phir agle step mein fetch layenge. Seedha fetch se shuru karna matlab — ek saath do cheezein samajhna. Pehle ek, phir doosra.

Toh abhi ka kaam: **hardcoded movies array banao — page pe dikkhao.**

---

## Step 1 — Movies Array Banao

`app/movies/page.tsx` kholo. Abhi yahan sirf ek h1 hai. 

Pehle socho — ek movie mein kya hota hai? Naam, year, rating. Toh ek object aisa dikhega:

```tsx
{ id: 1, title: "Inception", year: 2010, rating: 8.8 }
```

Ab aisa array banao page ke upar — component se bahar:

```tsx
const movies = [
  { id: 1, title: "Inception", year: 2010, rating: 8.8 },
  { id: 2, title: "Interstellar", year: 2014, rating: 8.6 },
  { id: 3, title: "The Dark Knight", year: 2008, rating: 9.0 },
]

export default function MoviesPage() {
  return (
    <div>
      <h1>Movies Page</h1>
    </div>
  )
}
```

**Verify karo:** Save karo — page pe koi change nahi dikhega abhi. Normal hai — array banaya, use nahi kiya abhi. ✅

---

## Step 2 — Array Ko Page Pe Dikkhao

Ab `movies` array ko page pe render karna hai.

`.map()` kya karta hai? — Array ka har element leta hai, usse ek cheez mein convert karta hai, naya array return karta hai.

```
[movie1, movie2, movie3] → [<div>movie1</div>, <div>movie2</div>, <div>movie3</div>]
```

Pehle sirf title dikhhao:

```tsx
export default function MoviesPage() {
  return (
    <div>
      <h1>Movies</h1>
      {movies.map((movie) => (
        <div key={movie.id}>
          <p>{movie.title}</p>
        </div>
      ))}
    </div>
  )
}
```

`key={movie.id}` — React ko track karne ke liye chahiye ki kaunsa element kaunsa hai. Har item unique key maangta hai.

**Verify karo:** Browser mein `/movies` kholo — teen movie titles dikhne chahiye. ✅

---

## Step 3 — Thoda Aur Info Dikhhao

Sirf title kafi nahi — year aur rating bhi dikhhao:

```tsx
{movies.map((movie) => (
  <div key={movie.id}>
    <p>{movie.title}</p>
    <p>{movie.year} • ⭐ {movie.rating}</p>
  </div>
))}
```

**Verify karo:** Har movie ke neeche year aur rating dikh rahi hai? ✅

---

## Step 4 — Console Mein Dekho

Kisi bhi naye concept se pehle console mein dekho kya chal raha hai.

Yeh ek **Server Component** hai — toh console browser mein nahi, **terminal** mein dikhega.

`MoviesPage` ke andar sabse pehle line mein yeh daalo:

```tsx
export default function MoviesPage() {
  console.log("Movies array:", movies)
  // ...
}
```

Terminal dekho — movies array print hogi.

Yeh important observation hai: **Server Component ka console terminal mein aata hai — browser console mein nahi.**

Dekh liya? Ab console.log hata do — kaam ho gaya.

---

## Summary

- Hardcoded `movies` array banaya
- `.map()` se array ko JSX mein convert kiya
- `key` prop kyun zaroori hai — samjha
- Server Component ka console terminal mein aata hai — confirm kiya

## Agla Step

Abhi movies hardcoded hain — directly component mein. Agla step: fetch karne ka **illusion** banao — ek async function jo data return kare jaise API se aa raha ho. Actual fetch nahi — sirf structure samajhna hai.
