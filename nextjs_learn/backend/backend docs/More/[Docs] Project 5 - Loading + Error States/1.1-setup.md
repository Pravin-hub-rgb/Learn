# Doc 1.1 — Next.js App Create Karo + Structure Samjho

## Abhi Tak Kya Hua
Bilkul fresh start — Project 5 shuru ho raha hai. Yeh project sirf ek cheez ke liye hai: **Loading aur Error States** — Next.js mein yeh kaise kaam karta hai, andar se samjhenge.

---

## Humara Overall Goal — Pehle Yeh Samjho

Is poore project mein hum ek Movies app banayenge.

Kyun movies? Kyunki movies fetch karne mein time lagta hai — network slow ho sakti hai — data na mile toh error aata hai. Yeh sab real problems hain jo hum solve karenge.

Par abhi 1.1 mein sirf **ek kaam** — app banana aur structure samajhna. Koi fetch nahi, koi loading nahi. Sirf: app chalao, structure dekho, samjho.

---

## Step 1 — Next.js App Banao

Terminal kholo aur yeh run karo:

```bash
npx create-next-app@latest project5-loading-error
```

Yeh poochega kuch sawaal — aisa jawab do:

```
Would you like to use TypeScript? → Yes
Would you like to use ESLint? → Yes
Would you like to use Tailwind CSS? → Yes
Would you like your code inside a `src/` directory? → No
Would you like to use App Router? → Yes
Would you like to use Turbopack? → Yes
Would you like to customize the import alias? → No
```

Install hone do. Phir:

```bash
cd project5-loading-error
npm run dev
```

Browser mein kholo: `http://localhost:3000`

Next.js ka default page dikhai dega. ✅

---

## Step 2 — Default Clutter Hatao

`app/page.tsx` kholo. Abhi wahan bahut saara default code hai — sab hata do. Sirf yeh rakho:

```tsx
export default function Home() {
  return (
    <div>
      <h1>Home Page</h1>
    </div>
  )
}
```

Save karo. Browser mein dekho — sirf "Home Page" dikhega.

**Verify karo:** Browser mein `http://localhost:3000` pe "Home Page" dikh raha hai? ✅

---

## Step 3 — App Ka Structure Samjho

Ab socho — hum movies app bana rahe hain. Hume ek `/movies` route chahiye.

Next.js mein route banana ka matlab hai — `app/` folder ke andar ek naya folder banana.

Toh `/movies` ke liye — `app/movies/` folder banao — andar `page.tsx` rakho.

```
app/
├── layout.tsx      ← Poori app ka wrapper (already hai)
├── page.tsx        ← / route (Home)
└── movies/
    └── page.tsx    ← /movies route (yeh abhi banayenge)
```

`app/movies/page.tsx` banao:

```tsx
export default function MoviesPage() {
  return (
    <div>
      <h1>Movies Page</h1>
    </div>
  )
}
```

Save karo. Browser mein `http://localhost:3000/movies` kholo.

**Verify karo:** "Movies Page" dikh raha hai? ✅

---

## Step 4 — Home Page Pe Link Daalo

`app/page.tsx` mein ek link daalo movies page ka. Pehle socho — HTML mein link kaise dalte hain? `<a href="/movies">` — haan, par Next.js mein uski jagah `<Link>` use karte hain.

Kyun `<Link>`? Kyunki `<a>` se poora page reload hota hai. `<Link>` se sirf woh part update hota hai jo badla — fast hota hai.

```tsx
import Link from 'next/link'

export default function Home() {
  return (
    <div>
      <h1>Home Page</h1>
      <Link href="/movies">Movies dekho →</Link>
    </div>
  )
}
```

`Link` kahan se aa raha hai? — `next/link` se. Yeh Next.js ka built-in component hai.

**Verify karo:** Home pe link dikh raha hai? Click karo — movies page pe jata hai? ✅
