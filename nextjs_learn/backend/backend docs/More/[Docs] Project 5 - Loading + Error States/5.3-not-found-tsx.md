# Doc 5.3 — `not-found.tsx` Banao — Custom 404 Page

## Abhi Tak Kya Hua
5.2 mein `notFound()` call kiya — Next.js ka default 404 page aa raha hai. Ab custom banana hai.

---

## `not-found.tsx` — Convention File

Same pattern — convention file — sahi jagah rakho — automatically kaam karega.

`notFound()` call hone pe — Next.js nearest `not-found.tsx` dhundhta hai.

"Nearest" matlab — pehle same folder mein dekho — nahi mila toh parent folder mein — nahi mila toh global (app/not-found.tsx).

---

## Step 1 — Movie-Specific `not-found.tsx`

`app/movies/[id]/` folder mein banao: `not-found.tsx`

```
app/
└── movies/
    └── [id]/
        ├── page.tsx
        └── not-found.tsx   ← naya
```

```tsx
export default function MovieNotFound() {
  return (
    <div>
      <h2>Movie Nahi Mili 🎬</h2>
      <p>Yeh movie exist nahi karti.</p>
    </div>
  )
}
```

**Verify karo:** `/movies/999` kholo — "Movie Nahi Mili 🎬" dikh raha hai? ✅

**Verify karo:** `/movies/1` normal movie detail page dikh raha hai? ✅

---

## Step 2 — Back Button Daalo

User stuck ho gaya 404 pe — wapas kaise jaaye:

```tsx
import Link from 'next/link'

export default function MovieNotFound() {
  return (
    <div>
      <h2>Movie Nahi Mili 🎬</h2>
      <p>Yeh movie exist nahi karti.</p>
      <Link href="/movies">← Wapas Movies Pe</Link>
    </div>
  )
}
```

**Verify karo:** Link kaam karta hai? `/movies` pe wapas jaata hai? ✅

---

## Step 3 — Global `not-found.tsx` Bhi Hota Hai

`app/not-found.tsx` — root level pe — poori app ke liye.

Agar `/kuch-bhi-random` kholo — koi route nahi — yeh global `not-found.tsx` show hota hai.

`app/not-found.tsx` banao:

```tsx
export default function GlobalNotFound() {
  return (
    <div>
      <h2>404 — Page Nahi Mila</h2>
      <p>Yeh page exist nahi karta.</p>
    </div>
  )
}
```

**Verify karo:** `http://localhost:3000/random-page` kholo — global not found dikhta hai? ✅

**Verify karo:** `/movies/999` pe movie-specific wala dikhta hai — global wala nahi? ✅

Next.js nearest file use karta hai — `[id]/not-found.tsx` zyada specific hai toh woh chalega.

---

## Summary

- `not-found.tsx` — custom 404 page
- `notFound()` call hone pe nearest `not-found.tsx` dhundha jaata hai
- `app/movies/[id]/not-found.tsx` — sirf movie detail ke liye
- `app/not-found.tsx` — global — poori app ke liye
- Nearest file wins

## Agla Step

Phase 5 complete. Phase 6: **Global Error** — root layout mein error aaye toh normal `error.tsx` kaam nahi karta — kyun aur solution kya hai.
