# Doc 5.2 — `notFound()` Function — Kab Call Karo

## Abhi Tak Kya Hua
5.1 mein problem dekhi — `/movies/999` pe blank page — data nahi mila par error bhi nahi. Manually handle karna hai.

---

## `notFound()` Kya Karta Hai

`notFound()` — `next/navigation` se import hota hai — call karo aur yeh:
1. HTTP 404 status set karta hai
2. `not-found.tsx` page render karta hai (agar hai)
3. Current component ka execution rok deta hai — `return` ki tarah

---

## Step 1 — Empty Check Daalo

`app/movies/[id]/page.tsx` mein pehle console.log se confirm karo kya aa raha hai:

```tsx
import { getMovieById } from '../lib'

export default async function MovieDetailPage({ 
  params 
}: { 
  params: Promise<{ id: string }> 
}) {
  const { id } = await params
  const movie = await getMovieById(id)
  
  console.log("ID:", id)
  console.log("Movie:", movie)
  console.log("Title exist karta hai?", movie.title)

  return (
    <div>
      <h1>{movie.title}</h1>
      <p>{movie.body}</p>
    </div>
  )
}
```

**Verify karo:** `/movies/999` — terminal mein `Title exist karta hai?` ke baad kya aaya? — `undefined` ✅

---

## Step 2 — `notFound()` Call Karo

Pehle import karo — jab zaroorat padi tab:

```tsx
import { notFound } from 'next/navigation'
import { getMovieById } from '../lib'

export default async function MovieDetailPage({ 
  params 
}: { 
  params: Promise<{ id: string }> 
}) {
  const { id } = await params
  const movie = await getMovieById(id)

  if (!movie.title) {
    notFound()
  }

  return (
    <div>
      <h1>{movie.title}</h1>
      <p>{movie.body}</p>
    </div>
  )
}
```

`!movie.title` — agar title nahi hai (undefined ya empty) — movie exist nahi karti.

**Verify karo:** `/movies/999` kholo — kya dikhta hai?

---

Next.js ka default 404 page dikhega — "404 | This page could not be found."

Yeh default hai — theek hai par hum custom banana chahte hain. ✅

**Verify karo:** `/movies/1` still kaam karta hai? ✅

## Summary

- `notFound()` — `next/navigation` se
- Call karo jab data nahi mila
- Execution rok deta hai — baaki code nahi chalta
- Default 404 page dikhhata hai — ya custom `not-found.tsx`

## Agla Step

Custom `not-found.tsx` banayenge — apna 404 page.
