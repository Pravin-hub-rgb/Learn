# Doc 2.1 — Real Fetch Karo — Phir Problem Dekho

## Abhi Tak Kya Hua
1.3 mein `lib.ts` banaya — `getMovies()` async function — hardcoded data return karta hai. Page mein `await` se call kiya.

---

## Abhi Ka Goal

Hardcoded data hatao — JSONPlaceholder se real fetch karo.

JSONPlaceholder ek free fake API hai — `https://jsonplaceholder.typicode.com/posts` — yeh posts deta hai. Hum posts ko movies ki tarah treat karenge.

---

## Step 1 — `fetch` Karo

`lib.ts` kholo. Hardcoded array hatao — uski jagah fetch karo:

```ts
export async function getMovies() {
  const response = await fetch('https://jsonplaceholder.typicode.com/posts')
  return response
}
```

Abhi sirf `response` return kar rahe hain — pura nahi. Pehle dekho kya aata hai.

`page.tsx` mein console.log update karo:

```tsx
const movies = await getMovies()
console.log("Response:", movies)
```

**Verify karo:** Terminal mein kya print hua? — Ek `Response` object dikhega — andar data nahi, sirf metadata. ✅

---

## Step 2 — Response Se Data Nikalo

`Response` object mein actual data nahi hota — `.json()` call karna padta hai usse:

```ts
export async function getMovies() {
  const response = await fetch('https://jsonplaceholder.typicode.com/posts')
  const data = await response.json()
  return data
}
```

`.json()` bhi async hai — isliye `await` lagaya.

**Verify karo:** Terminal mein ab 100 posts ka array aayega. ✅

---

## Step 3 — Sirf 10 Lo — Aur Map Fix Karo

100 posts bahut hain — sirf pehle 10 lo. `.slice(0, 10)` karta hai: index 0 se 10 tak — 10 elements.

```ts
export async function getMovies() {
  const response = await fetch('https://jsonplaceholder.typicode.com/posts')
  const data = await response.json()
  return data.slice(0, 10)
}
```

Ab `page.tsx` mein `.map()` fix karo — posts ke fields alag hain — `id`, `title`, `body`:

```tsx
{movies.map((movie: any) => (
  <div key={movie.id}>
    <p>{movie.title}</p>
    <p>{movie.body}</p>
  </div>
))}
```

**Verify karo:** Browser mein 10 posts dikh rahi hain? ✅

---

## Step 4 — Delay Daalo — Problem Dekho

Abhi fetch fast hai — local machine pe internet fast hai. Real world mein slow network hoti hai.

`lib.ts` mein artificially delay daalo:

```ts
export async function getMovies() {
  await new Promise((resolve) => setTimeout(resolve, 3000))
  
  const response = await fetch('https://jsonplaceholder.typicode.com/posts')
  const data = await response.json()
  return data.slice(0, 10)
}
```

`new Promise(...)` wala line ek 3 second ka ruk jaao command hai. `setTimeout` 3000 milliseconds = 3 seconds baad resolve karta hai.

**Verify karo:** Browser mein `/movies` refresh karo — **3 second tak page bilkul blank** — phir data aata hai.

Yeh problem hai. 😬

User ko kuch nahi dikh raha — koi feedback nahi — kya chal raha hai? Load ho raha hai? Crash ho gaya? Pata hi nahi.

---

## Summary

- Real fetch kiya — JSONPlaceholder se
- `response.json()` se actual data nikala
- `.slice(0, 10)` se sirf 10 items liye
- Delay daala — **blank page problem** dekhI

## Agla Step

Yeh blank page problem fix karni hai. Next.js mein iske liye ek special file hoti hai — `loading.tsx` — jo automatically show hoti hai jab page data load kar raha ho. Woh banayenge agla step mein.
