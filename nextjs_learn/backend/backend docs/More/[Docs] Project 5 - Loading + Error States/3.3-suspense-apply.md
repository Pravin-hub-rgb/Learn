## Step 2 ‚Äî Fetch `MoviesList` Mein Le Jao

Ab `getMovies` call is component mein aayegi ‚Äî component ko async banana padega:

```tsx
import { getMovies } from './lib'

export default async function MoviesList() {
  const movies = await getMovies()

  return (
    <div>
      {movies.map((movie: any) => (
        <div key={movie.id}>
          <p>{movie.title}</p>
          <p>{movie.body}</p>
        </div>
      ))}
    </div>
  )
}
```

**Verify karo:** File save karo. Abhi `page.tsx` mein use nahi kiya ‚Äî woh agle step mein. ‚úÖ

## Step 3 ‚Äî `page.tsx` Update Karo

`app/movies/page.tsx` mein `getMovies` ka call hatao ‚Äî aur `MoviesList` import karo:

```tsx
import { Suspense } from 'react'
import MoviesList from './MoviesList'

export default function MoviesPage() {
  return (
    <div>
      <div>
        <h1>Movies</h1>
        <p>Yahan movies hain:</p>
      </div>

      <Suspense fallback={<p>Movies load ho rahi hain...</p>}>
        <MoviesList />
      </Suspense>
    </div>
  )
}
```

Dhyan do:
- `MoviesPage` ab `async` nahi ‚Äî koi await nahi isme
- `Suspense` ‚Äî `react` se import kiya
- `<MoviesList />` ‚Äî Suspense ke andar ‚Äî woh async hai ‚Äî Suspense handle karega

**Verify karo:** `/movies` refresh karo ‚Äî **Header turant dikhe** ‚Äî phir 3 second baad movies list dikhe. ‚úÖ

Yeh problem fix ho gayi!
