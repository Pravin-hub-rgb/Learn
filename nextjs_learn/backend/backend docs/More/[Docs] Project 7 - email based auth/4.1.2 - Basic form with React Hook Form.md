# Doc 4.1.2 — Login Form: React Hook Form Basics

## Abhi Tak Kya Hua (4.1.1)

- `/login` route ban gaya
- Sirf `<h1>Login</h1>` heading thi
- Koi form nahi tha

## Aaj Ka Goal (4.1.2)

**Aaj hum banayenge:** Simple login form with React Hook Form

```
Email input + Password input + Submit button
Form submit par console.log (abhi server action nahi)
```

---

## Step 1: Problem Samjho — "Input Ko Handle Karna" Kya Hai?

**Soch:** "Mujhe login form banana hai. Lekin 'form banana' ka matlab sirf HTML inputs add karna nahi hai. Mujhe yeh bhi handle karna hai:"

**Problems jo solve karni hain:**

1. **User ne email field mein kya type kiya?** — Woh value capture karni hai
2. **User ne password field mein kya type kiya?** — Woh bhi capture karni hai
3. **Form submit hua toh kya hoga?** — Dono values ko lekar kuch karna hai (jaise console.log karna, ya server par bhejna)
4. **Validation:** Email khali toh nahi? Password kam se kam 8 characters ka hai?
5. **Loading state:** Form submit ho raha hai toh button disable karna hai, "Submitting..." dikhana hai

**Yeh sab "form state management" kehlata hai.**

**Options:**

**Option 1: Manual (useState)**
```tsx
const [email, setEmail] = useState("");
const [password, setPassword] = useState("");

// Har input par onChange handler
<input 
  value={email} 
  onChange={(e) => setEmail(e.target.value)} 
/>
```
Problem: Har input ke liye alag state, alag onChange — code bahut ho jata hai.

**Option 2: React Hook Form (Better)**
```tsx
const { register } = useForm();
<input {...register("email")} />
```
Sab kuch automatically handle hota hai — ek line mein!

**Decision:** React Hook Form use karenge kyunki:
- Code kam likhna padta hai
- Performance better hai
- Validation integrate karna easy hai
- Loading state, error handling built-in hai

## Step 3: `useForm` Hook Import Karna

**Soch:** "Ab React Hook Form ka `useForm` hook import karna hai. Yeh hook form ka saara state manage karta hai — inputs ka data, validation errors, loading state, sab kuch."

**Import statement:**
```tsx
import { useForm } from "react-hook-form";
```

**`useForm` kya return karta hai:**
- `register` — Inputs ko form fields se connect karne ke liye
- `handleSubmit` — Form submission handle karne ke liye
- `formState` — Errors, loading state, etc.
- Aur bhi cheezein (reset, watch, etc.)

---

## Step 4: Pehla Step — Sirf `register` Use Karna

**Soch:** "Sabse pehle mujhe inputs ko form se connect karna hai. Iske liye `register` function use hota hai. Main pehle sirf `register` hi use karunga — baaki cheezein baad mein add karunga."

**`register` kya karta hai:**
- Input ko form field se connect karta hai
- Automatically `onChange`, `onBlur`, `ref` set kar deta hai
- Input ka data form ke saath track hota hai
- Jab user type karta hai, form automatically data save karta hai

**Code (sirf `register` ke saath):**
```tsx
"use client";

import { useForm } from "react-hook-form";

export default function LoginPage() {
  // Pehle sirf register le rahe hain
  const { register } = useForm();

  return (
    <div>
      <h1>Login</h1>
      <input type="email" {...register("email")} placeholder="Enter email" />
      <input type="password" {...register("password")} placeholder="Enter password" />
    </div>
  );
}
```

**Explanation:**
- `const { register } = useForm()` — `useForm` hook call kiya, sirf `register` function nikala
- `{...register("email")}` — Spread operator se `register("email")` ke saare properties input par laga diye
  - Yeh automatically `onChange`, `onBlur`, `ref` set kar deta hai
  - Input ab "email" field se connect ho gaya

**Test karo:**
1. Browser mein jao: `http://localhost:3000/login`
2. Email aur password enter karo
3. Inputs kaam kar rahe hain — data type kar sakte hain

**Lekin abhi problem hai:**
- Form submit karne ka koi tarika nahi hai
- Button nahi hai
- Submit handler nahi hai
- Data capture toh ho raha hai, lekin use kaise karein?

## Step 5: Submit Button Add Karna

**Soch:** "Ab mujhe form submit karne ke liye button add karna hai. Button par click karne par form submit hoga."

**Code (button add karo):**
```tsx
"use client";

import { useForm } from "react-hook-form";

export default function LoginPage() {
  const { register } = useForm();

  return (
    <div>
      <h1>Login</h1>
      <input type="email" {...register("email")} placeholder="Enter email" />
      <input type="password" {...register("password")} placeholder="Enter password" />
      <button type="submit">Login</button>
    </div>
  );
}
```

**Test karo:**
1. Form mein data enter karo
2. "Login" button par click karo
3. **Problem:** Page refresh ho jata hai! (default form submission behavior)

**Kyun hua yeh?**
- HTML form ka default behavior hai ki submit button click karne par page refresh ho jata hai
- Yeh isliye hota hai kyunki traditionally form data server par bhejne ke liye page refresh hota tha
- Lekin humein page refresh nahi chahiye — humein data ko JavaScript mein handle karna hai

## Step 6: Problem — Form Submit Handle Kaise Karein?

**Soch:** "Button click karne par page refresh ho raha hai. Mujhe form submission handle karna padega. Traditional approach mein `onSubmit` handler use karte hain aur `e.preventDefault()` call karte hain."

**Traditional approach (bina `handleSubmit` ke):**
```tsx
const handleSubmit = (e: React.FormEvent) => {
  e.preventDefault(); // Page refresh rokne ke liye
  // Ab form data kaise nikalu?
  // Mujhe manually inputs se values nikalni padengi!
};
```

**Problem:** Form data nikalne ke liye har input ko manually track karna padega — yeh messy hai. Ya toh `useState` use karna padega, ya DOM se values nikalni padengi.

**Solution:** React Hook Form ka `handleSubmit` use karna — woh automatically:
1. `preventDefault()` call karta hai (page refresh nahi hota)
2. Form data collect karta hai
3. Hamare `onSubmit` function ko data pass karta hai

---

## Step 7: `handleSubmit` Introduce Karna

**Soch:** "Ab main `handleSubmit` use karunga jo automatically page refresh rokega aur form data collect karega."

**`handleSubmit` kya karta hai:**
- Automatically `e.preventDefault()` call karta hai (page refresh nahi hota)
- Form data collect karta hai — saare registered inputs ka data ek object mein
- Hamare `onSubmit` function ko data pass karta hai
- Validation bhi handle karta hai (baad mein add karenge)

**Pehle `handleSubmit` aur `onSubmit` handler add karo:**
```tsx
"use client";

import { useForm } from "react-hook-form";

export default function LoginPage() {
  const { register, handleSubmit } = useForm();

  // Yeh function tab call hoga jab form submit hoga
  const onSubmit = (data: any) => {
    console.log("Form submitted:", data);
  };

  return (
    <div>
      <h1>Login</h1>
      <input type="email" {...register("email")} placeholder="Enter email" />
      <input type="password" {...register("password")} placeholder="Enter password" />
      <button type="submit" onClick={handleSubmit(onSubmit)}>Login</button>
    </div>
  );
}
```

**Wait — yeh galat hai!** `onClick` use nahi karna chahiye — form tag par `onSubmit` use karna chahiye.

**Sahi approach:** Inputs ko `<form>` tag mein wrap karo aur form par `onSubmit={handleSubmit(onSubmit)}` lagao.

---

## Step 8: Form Tag Mein Wrap Karna (Sahi Approach)

**Soch:** "Sahi approach yeh hai ki inputs ko `<form>` tag mein wrap karu aur form par `onSubmit={handleSubmit(onSubmit)}` lagau."

**Final code:**
```tsx
"use client";

import { useForm } from "react-hook-form";

export default function LoginPage() {
  const { register, handleSubmit } = useForm();

  const onSubmit = (data: any) => {
    console.log("Form submitted:", data);
  };

  return (
    <div>
      <h1>Login</h1>
      <form onSubmit={handleSubmit(onSubmit)}>
        <input type="email" {...register("email")} placeholder="Enter email" />
        <input type="password" {...register("password")} placeholder="Enter password" />
        <button type="submit">Login</button>
      </form>
    </div>
  );
}
```

**Kya hua:**
- `<form onSubmit={handleSubmit(onSubmit)}>` — Form tag par `handleSubmit` laga diya
- Ab jab button click hoga:
  1. Form submit hoga
  2. `handleSubmit` automatically `preventDefault()` call karega (page refresh nahi hoga)
  3. Form data collect karega — `{ email: "...", password: "..." }`
  4. `onSubmit(data)` function call karega

## Summary

**Aaj humne kya seekha (step by step):**

1. **Problem samjhi:** "Input ko handle karna" ka matlab hai — user ka type kiya hua data capture karna, validate karna, submit karna

2. **React Hook Form choose kiya** kyunki manual approach messy hai

3. **`useForm` import kiya** — hook use karne ke liye

4. **Pehle sirf `register` use kiya** — inputs ko form se connect karne ke liye
   - `{...register("email")}` — Input ko "email" field se connect kiya

5. **Button add kiya** — form submit karne ke liye

6. **Problem aaya** — page refresh ho raha hai

7. **`handleSubmit` introduce kiya** — automatic `preventDefault()` aur data collection

8. **Form tag mein wrap kiya** — sahi approach se sab kuch connect kiya

**Key concepts:**
- `register("fieldName")` — Input ko form field se connect karta hai, automatically onChange/onBlur/ref set kar deta hai
- `handleSubmit(onSubmit)` — Form submission handle karta hai (auto preventDefault, data collection)
- `onSubmit(data)` — Hamara custom handler jisme form data milta hai

**Abhi kya nahi kiya:**
- Validation nahi kiya (RHF se karenge 4.1.3 mein)
- Server action nahi banaya (baad mein aayega)
- Sirf console.log kiya
