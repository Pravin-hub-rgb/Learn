## Password Reset System Ka Poora Flow

```
User login page par jata hai → "Forgot Password?" par click karta hai
  ↓
Email input wala page dikhta hai
  ↓
User apna email daal ke submit karta hai
  ↓
✅ Hum backend mein check karte hai ki yeh email exist karta hai ya nahi
  ↓
✅ Agar email exist karta hai:
    - Ek unique random token generate karo
    - Token ko database mein save karo 15 minute ke expiry ke saath
    - Email bhejo us email par reset password ke link ke saath
    - User ko page par dikhao: "Check your email"
  ↓
User email kholta hai → Reset password link par click karta hai
  ↓
Naya password wala page dikhta hai
  ↓
User naya password daal ke submit karta hai
  ↓
✅ Hum check karte hai:
    - Token valid hai ya nahi
    - Token expire toh nahi hua
    - Token kisi user ka hai ya nahi
  ↓
✅ Agar sab sahi hai:
    - Naya password hash karo
    - User ke password ko update karo database mein
    - Us user ke saare purane sessions ko delete karo
    - Token ko delete karo
    - Login page par redirect karo
    - Success message dikhao: "Password reset successfully! Login karo"
```

---

## Har Step Ki Deep Explanation

### ✅ Step 1: Forgot Password Button

Login page par "Forgot Password?" ka link hoga. User yaha click karega toh woh `/forgot-password` page par jayega.

### ✅ Step 2: Email Input Form

Is page par sirf ek input hoga email ka. User apna registered email daalega.

### ✅ Step 3: Backend Check

Hum backend mein pehle check karenge ki yeh email exist karta hai ya nahi.

> 🧠 Important:
> Chahe email exist kare ya nahi hum user ko hamesha same message dikhayenge: "Check your email"
> Kyunki hum nahi batayenge ki yeh email exist karta hai ya nahi. Security risk hai.

### ✅ Step 4: Token Generate Karo

Agar email exist karta hai toh hum ek unique random token generate karenge 32 characters ka.

Is token mein hum store karenge:
- Konsa user hai
- Kitne time baad ye token expire hoga (15 minutes)

### ✅ Step 5: Email Bhejo

Us email par ek link bhejenge:
`https://yourapp.com/reset-password?token=abc123xyz`

### ✅ Step 6: Reset Password Page

User link par click karega toh woh `/reset-password` page par aayega token ke saath.

Is page par 2 input honge:
1.  Naya Password
2.  Confirm Password

### ✅ Step 7: Token Verify Karo

Hum backend mein check karenge:
- Token database mein hai ya nahi
- Token expire toh nahi hua
- Token kisi user se associated hai ya nahi

### ✅ Step 8: Password Update Karo

Agar sab sahi hai toh:
1.  Naya password ko hash karo
2.  User ke password ko database mein update karo
3.  **Us user ke saare sessions delete karo**
    > 🧠 Kyun?
    > Agar kisi ne already user ke account mein login kiya hua hai toh usko bhi logout kar do. Security ke liye.
4.  Token ko database se delete karo (ek baar use ho gaya hai)
5.  User ko login page par redirect karo
