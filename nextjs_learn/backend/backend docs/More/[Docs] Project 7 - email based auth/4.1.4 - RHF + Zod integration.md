## Step 2: zodResolver Use Karna

**Soch:** "Ab `zodResolver` use karke Zod schema ko RHF ke `useForm` hook ke saath connect karna hai. `resolver` option mein `zodResolver(loginSchema)` pass karunga."

**`zodResolver` kya karta hai:**
- Zod schema ko RHF ke saath integrate karta hai
- Form submit hone se pehle automatically validate karta hai
- Validation errors ko `formState.errors` mein populate karta hai

```tsx
"use client";

import { useForm } from "react-hook-form";
import { zodResolver } from "@hookform/resolvers/zod";
import { loginSchema } from "@/lib/validations/auth";

export default function LoginPage() {
  const { register, handleSubmit, formState: { errors } } = useForm({
    resolver: zodResolver(loginSchema),
  });

  const onSubmit = (data: any) => {
    console.log("Form submitted:", data);
  };

  return (
    <div>
      <h1>Login</h1>
      <form onSubmit={handleSubmit(onSubmit)}>
        <input type="email" {...register("email")} />
        {errors.email && <p>{errors.email.message}</p>}

        <input type="password" {...register("password")} />
        {errors.password && <p>{errors.password.message}</p>}

        <button type="submit">Login</button>
      </form>
    </div>
  );
}
```

**Explanation:**

- `resolver: zodResolver(loginSchema)` — `useForm` ke options mein `resolver` pass karte hain jo Zod schema se validation perform karta hai
- `formState: { errors }` — `formState` se validation errors extract karte hain. `errors` object mein har field ka error message hota hai
- `{errors.email && <p>{errors.email.message}</p>}` — Conditional rendering se error tab dikhta hai jab validation fail hoti hai
- `errors.email.message` — Zod schema mein define kiya hua error message

## Complete Code

```tsx
// app/(auth)/login/page.tsx

"use client";

import { useForm } from "react-hook-form";
import { zodResolver } from "@hookform/resolvers/zod";
import { loginSchema } from "@/lib/validations/auth";

export default function LoginPage() {
  const { register, handleSubmit, formState: { errors } } = useForm({
    resolver: zodResolver(loginSchema),
  });

  const onSubmit = (data: any) => {
    console.log("Form submitted:", data);
  };

  return (
    <div>
      <h1>Login</h1>
      <form onSubmit={handleSubmit(onSubmit)}>
        <div>
          <input
            type="email"
            placeholder="Enter your email"
            {...register("email")}
          />
          {errors.email && (
            <p style={{ color: "red" }}>{errors.email.message}</p>
          )}
        </div>

        <div>
          <input
            type="password"
            placeholder="Enter your password"
            {...register("password")}
          />
          {errors.password && (
            <p style={{ color: "red" }}>{errors.password.message}</p>
          )}
        </div>

        <button type="submit">Login</button>
      </form>
    </div>
  );
}
```

## Common Issues

### Issue 1: "zodResolver is not defined"

**Check:**
- `import { zodResolver } from "@hookform/resolvers/zod";` add kiya?
- `@hookform/resolvers` install kiya?

### Issue 2: Errors object undefined hai

**Check:**
- `formState: { errors }` destructured kiya?

### Issue 3: Form phir bhi submit ho raha hai invalid data ke saath

**Check:**
- `handleSubmit(onSubmit)` use kar rahe ho na?
- `resolver` sahi se set hai `useForm` mein?
