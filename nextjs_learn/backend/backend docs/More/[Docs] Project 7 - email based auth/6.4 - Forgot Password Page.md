### ✅ Step 2.3 Ab page refresh nahi karna hai

🧠 **SOCH:**
Ab tum submit karoge toh poora page refresh ho jata hai. Hume nahi chahiye poora page refresh. Hume sirf background mein request bhejna hai.

Isliye hum `useActionState` hook use karenge.

📝 **CODE:**
```tsx
"use client";

import { useActionState } from "react";

export default function ForgotPasswordPage() {
  const [state, formAction] = useActionState(() => {
    console.log("Button click kiya hai!");
  }, null);

  return (
    <div className="max-w-md mx-auto mt-20 p-6 border rounded-lg">
      <h1 className="text-2xl font-bold mb-6">Forgot Password</h1>

      <form action={formAction}>
        <div>
          <label>Email</label>
          <input
            type="email"
            name="email"
            required
          />
        </div>

        <button type="submit">
          Send Reset Link
        </button>
      </form>
    </div>
  );
}
```

📚 **EXPLANATION:**
1.  `"use client"`:
    - Yeh lagana padta hai kyunki `useActionState` browser mein chalta hai
    - Iske bina tum hook nahi use kar sakte

2.  `useActionState`:
    - Yeh ek hook hai React ka
    - Isse tum form submit karte waqt page refresh nahi hoga
    - Bas itna hi kaam hai abhi ke liye

3.  `() => { console.log("Button click kiya hai!") }`:
    - Abhi yeh sirf dummy function hai
    - Abhi ke liye bas console mein print karega
    - Agle doc mein jab hum server action banayenge tab iski jagah actual action aayega

4.  `<form action={formAction}>`:
    - Ab tum form mein `action` mein `formAction` pass karo
    - Ab tum submit karoge toh page refresh nahi hoga ✅
