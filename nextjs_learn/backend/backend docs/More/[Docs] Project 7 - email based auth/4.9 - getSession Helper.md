## Step 1: Hum Kya Banane Wale Hain?

Ab hum aisa function banayenge jo:
1.  Request se cookie mein se `sessionId` niklega
2.  Database mein us session ID ko dhudega
3.  Check karega ki session expire toh nahi hua
4.  Agar sab sahi hai toh user return karega
5.  Agar kuch galat hai toh `null` return karega

Yeh function hum har jagah use karenge jahan hume current logged in user chahiye.

## Step 3: getSession Function Step by Step Banaye

Hum function ko step by step banayenge, har step mein thoda code add karenge uske baad explain karenge.

---

### Step 3.1: Function start karo aur cookie nikalo

Pehle sirf itna likho:

```typescript
export async function getSession() {
  const cookieStore = await cookies();
  const sessionId = cookieStore.get("sessionId")?.value;
}
```

> 🧠 Kya ho raha hai:
> - Humne `cookies()` ko await kiya hai Next.js 15 ke liye
> - `cookieStore.get("sessionId")` se hum cookie extract kar rahe hai
> - `?.value` optional chaining hai, agar cookie nahi hai toh error nahi aayega
> - Agar cookie hai toh uski value `sessionId` mein aa jayegi

---

### Step 3.2: Check karo session ID exist karta hai ya nahi

Ab thoda aur code add karo:

```typescript
export async function getSession() {
  const cookieStore = await cookies();
  const sessionId = cookieStore.get("sessionId")?.value;

  if (!sessionId) {
    return null;
  }
}
```

> 🧠 Kya ho raha hai:
> - Agar `sessionId` nahi hai matlab user ka cookie nahi hai
> - Matlab woh login nahi hai
> - Isliye direct `null` return kar dete hai, aage check karne ka koi fayda nahi hai

---

### Step 3.3: Database mein session dhundho

Ab aur code add karo:

```typescript
export async function getSession() {
  const cookieStore = await cookies();
  const sessionId = cookieStore.get("sessionId")?.value;

  if (!sessionId) {
    return null;
  }

  const session = await prisma.session.findUnique({
    where: { id: sessionId },
    include: { user: true }
  });
}
```

> 🧠 Kya ho raha hai:
> - `prisma.session.findUnique` se database mein session dhund rahe hai
> - `where: { id: sessionId }` se wahi session dhundega jo cookie mein hai
> - ✅ `include: { user: true }` isliye lagaya hai taaki saath mein user ka poora data bhi mil jaye
> - Humko baad mein user hi chahiye isliye abhi se hi include kar lete hai

---

### Step 3.4: Check karo session mila ya nahi

Ab aur code add karo:

```typescript
export async function getSession() {
  const cookieStore = await cookies();
  const sessionId = cookieStore.get("sessionId")?.value;

  if (!sessionId) {
    return null;
  }

  const session = await prisma.session.findUnique({
    where: { id: sessionId },
    include: { user: true }
  });

  if (!session) {
    return null;
  }
}
```

> 🧠 Kya ho raha hai:
> - Ho sakta hai cookie mein session ID hai lekin woh database mein nahi hai
> - Koi invalid session ID diya ho ya session already delete ho gaya ho
> - Is case mein bhi user login nahi hai, null return kar do

---

### Step 3.5: Expiry check karo

Ab aur code add karo:

```typescript
export async function getSession() {
  const cookieStore = await cookies();
  const sessionId = cookieStore.get("sessionId")?.value;

  if (!sessionId) {
    return null;
  }

  const session = await prisma.session.findUnique({
    where: { id: sessionId },
    include: { user: true }
  });

  if (!session) {
    return null;
  }

  if (session.expiresAt < new Date()) {
    await prisma.session.delete({ where: { id: sessionId } });
    return null;
  }
}
```

> 🧠 Kya ho raha hai:
> - Har session ka ek expiry time hota hai
> - Hum check kar rahe hai ki woh time abhi se peeche toh nahi hai
> - Agar session expire ho gaya hai toh usko database se bhi delete kar dete hai
> - Isse bakwaas stale sessions database mein nahi bachenge

## Step 5: Ab Poora session.ts file yeh dikhta hai

✅ Final `lib/session.ts` code:
```typescript
import prisma from "./prisma";
import { cookies } from "next/headers";

export async function createSession(userId: string) {
  const expiresAt = new Date(Date.now() + 7 * 24 * 60 * 60 * 1000);

  const session = await prisma.session.create({
    data: {
      userId,
      expiresAt,
    },
  });

  const cookieStore = await cookies();

  cookieStore.set("sessionId", session.id, {
    httpOnly: true,
    secure: process.env.NODE_ENV === "production",
    sameSite: "lax",
    expires: expiresAt,
    path: "/",
  });

  return session;
}

export async function getSession() {
  const cookieStore = await cookies();
  const sessionId = cookieStore.get("sessionId")?.value;

  if (!sessionId) {
    return null;
  }

  const session = await prisma.session.findUnique({
    where: { id: sessionId },
    include: { user: true }
  });

  if (!session) {
    return null;
  }

  if (session.expiresAt < new Date()) {
    await prisma.session.delete({ where: { id: sessionId } });
    return null;
  }

  return session.user;
}
```
