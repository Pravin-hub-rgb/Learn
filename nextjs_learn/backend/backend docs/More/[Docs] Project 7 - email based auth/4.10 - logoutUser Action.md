# Doc 4.10 — logoutUser Action

## Abhi Tak Kya Hua

- User login kar sakta hai
- Session create ho jata hai
- Hum getSession se user check kar sakte hai
- ✅ **Ab hum user ko logout karne ka system banayenge**

## Aaj Ka Goal (4.10)

✅ `logoutUser()` action banayenge
✅ User ke logout par session delete karenge
✅ Cookie clear karenge
✅ User ko login page par redirect karenge

---

## Step 1: Pehle deleteSession Helper Banaye

✅ Ab `lib/session.ts` file open karo. Uske end mein yeh naya function add karo:

```typescript
export async function deleteSession() {
  const cookieStore = await cookies();
  const sessionId = cookieStore.get("sessionId")?.value;

  if (sessionId) {
    // Database se session delete karo
    await prisma.session.delete({
      where: { id: sessionId }
    }).catch(() => {});
  }

  // Cookie se bhi sessionId delete karo
  cookieStore.delete("sessionId");
}
```

> 🧠 Kya ho raha hai:
> - Hum pehle cookie se sessionId nikal rahe hai
> - Agar sessionId hai toh database se us session ko delete kar rahe hai
> - `catch(() => {})` isliye lagaya hai agar session already delete ho gaya hai toh error nahi aayega
> - Last mein cookie se hi sessionId ko delete kar dete hai

---

## Step 2: logoutUser Action Banaye

✅ Ab `app/actions/auth.ts` file open karo. Usko end mein yeh naya action add karo:

```typescript
"use server";

import { deleteSession } from "@/lib/session";
import { redirect } from "next/navigation";

export async function logoutUser() {
  await deleteSession();
  redirect("/login");
}
```

> 🧠 Kya ho raha hai:
> - Humne `deleteSession` helper ko call kiya
> - Woh database se session delete kar dega
> - Woh cookie bhi clear kar dega
> - Phir user ko login page par redirect kar dete hai

---

## Step 3: Ab Logout Button Add Karo

✅ Ab kisi bhi page par tum logout button aise add kar sakte ho:

```tsx
import { logoutUser } from "@/app/actions/auth";

export default function Dashboard() {
  return (
    <form action={logoutUser}>
      <button type="submit">Logout</button>
    </form>
  );
}
```

> ✅ Yeh Next.js 15 mein Server Action ko directly form action mein use karne ka tareeqa hai.
> Button par click karte hi user logout ho jayega.
