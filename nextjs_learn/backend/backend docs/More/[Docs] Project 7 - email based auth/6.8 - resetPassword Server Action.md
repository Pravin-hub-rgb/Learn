### ✅ Step 2.3 Ab token verify karo

🧠 **SOCH:**
✅ Ab hume database mein yeh token exist karta hai ya nahi check karna hai
✅ Uska expiry time bhi check karna hai ki woh abhi tak valid hai ya nahi
✅ Agar token nahi mila ya expired hai toh error return karo

📝 **CODE:**
```typescript
"use server";

import prisma from "@/lib/prisma";

export async function resetPassword(prevState: unknown, formData: FormData) {
  const token = formData.get("token") as string;
  const password = formData.get("password") as string;
  const confirmPassword = formData.get("confirmPassword") as string;

  // Find token in database
  const passwordResetToken = await prisma.passwordResetToken.findUnique({
    where: { token }
  });

  if (!passwordResetToken) {
    return {
      error: "Invalid reset token"
    };
  }

  // Check if token is expired
  if (passwordResetToken.expiresAt < new Date()) {
    return {
      error: "Reset token has expired"
    };
  }

  console.log("Token is valid");

  return {};
}
```

### ✅ Step 2.5 Ab naya password hash karo aur save karo

🧠 **SOCH:**
✅ Ab hum naya password hash karenge
✅ User ke password ko update karenge database mein
✅ Token ko delete kar denge

📝 **CODE:**
```typescript
"use server";

import prisma from "@/lib/prisma";
import bcrypt from "bcryptjs";

export async function resetPassword(prevState: unknown, formData: FormData) {
  const token = formData.get("token") as string;
  const password = formData.get("password") as string;
  const confirmPassword = formData.get("confirmPassword") as string;

  const passwordResetToken = await prisma.passwordResetToken.findUnique({
    where: { token }
  });

  if (!passwordResetToken) {
    return {
      error: "Invalid reset token"
    };
  }

  if (passwordResetToken.expiresAt < new Date()) {
    return {
      error: "Reset token has expired"
    };
  }

  if (password.length < 8) {
    return {
      error: "Password must be at least 8 characters"
    };
  }

  if (password !== confirmPassword) {
    return {
      error: "Passwords do not match"
    };
  }

  // Hash new password
  const hashedPassword = await bcrypt.hash(password, 12);

  // Update user password
  await prisma.user.update({
    where: { id: passwordResetToken.userId },
    data: {
      password: hashedPassword
    }
  });

  // Delete token so it can't be used again
  await prisma.passwordResetToken.delete({
    where: { id: passwordResetToken.id }
  });

  return {
    success: true,
    message: "Password reset successfully. You can now login with your new password."
  };
}
```

### ✅ Step 2.6 Ab saare sessions invalidate karo

🧠 **SOCH:**
✅ Jab user password change karta hai toh uski saari purani sessions ko nikal dena chahiye
✅ Kyunki agar kisi ne uska password leke login kiya hua hai toh usko bhi logout kar dena chahiye
✅ Ye bahut important security step hai

📝 **CODE:**
```typescript
"use server";

import prisma from "@/lib/prisma";
import bcrypt from "bcryptjs";

export async function resetPassword(prevState: unknown, formData: FormData) {
  const token = formData.get("token") as string;
  const password = formData.get("password") as string;
  const confirmPassword = formData.get("confirmPassword") as string;

  const passwordResetToken = await prisma.passwordResetToken.findUnique({
    where: { token }
  });

  if (!passwordResetToken) {
    return {
      error: "Invalid reset token"
    };
  }

  if (passwordResetToken.expiresAt < new Date()) {
    return {
      error: "Reset token has expired"
    };
  }

  if (password.length < 8) {
    return {
      error: "Password must be at least 8 characters"
    };
  }

  if (password !== confirmPassword) {
    return {
      error: "Passwords do not match"
    };
  }

  // Hash new password
  const hashedPassword = await bcrypt.hash(password, 12);

  // Update user password
  await prisma.user.update({
    where: { id: passwordResetToken.userId },
    data: {
      password: hashedPassword
    }
  });

  // Invalidate all sessions for this user
  await prisma.session.deleteMany({
    where: { userId: passwordResetToken.userId }
  });

  // Delete token so it can't be used again
  await prisma.passwordResetToken.delete({
    where: { id: passwordResetToken.id }
  });

  return {
    success: true,
    message: "Password reset successfully. You can now login with your new password."
  };
}
```

> 🧠 Important:
> Hum saare sessions delete kar rahe hai kyunki agar koi dusra insaan is account mein login kiya hua hai toh usko ab automatic logout kar diya jayega. Jab password change hota hai toh ye karna hi chahiye.
