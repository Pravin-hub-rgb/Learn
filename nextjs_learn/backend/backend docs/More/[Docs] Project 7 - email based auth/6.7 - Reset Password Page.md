### ✅ Step 2.2 Ab URL se token nikalo

🧠 **SOCH:**
✅ Humne jo token email mein bheja tha woh URL mein query parameter mein hoga
✅ Hum us token ko nikalenge
✅ Next.js mein hum `useSearchParams()` hook se nikal sakte hai
✅ Agar token nahi hai toh link invalid hai

📝 **CODE:**
```tsx
"use client";

import { useSearchParams } from "next/navigation";

export default function ResetPasswordPage() {
  const searchParams = useSearchParams();
  const token = searchParams.get("token");

  return (
    <div className="max-w-md mx-auto mt-20 p-6 border rounded-lg">
      <h1 className="text-2xl font-bold mb-6">Reset Password</h1>

      <p>Token: {token}</p>
    </div>
  );
}
```

📚 **EXPLANATION:**
1.  `"use client"`:
    - `useSearchParams` client side hook hai isliye ye lagana padta hai
    - Isse page client component ban jata hai

2.  `useSearchParams()`:
    - Yeh Next.js ka inbuilt hook hai
    - Isse tum URL ke saare query parameters nikal sakte ho
    - Hum yaha se `token` nikal rahe hai

3.  `searchParams.get("token")`:
    - Yeh hume token ki value dega jo URL mein hai
    - Agar token nahi hai toh yeh `null` return karega

✅ Ab tum `/reset-password?token=abc123` par jao. Token dikh raha hai? ✅

### ✅ Step 2.5 Ab useActionState add karo

🧠 **SOCH:**
✅ Ab hum form submit karne par page refresh nahi karna hai
✅ Isliye `useActionState` hook use karenge
✅ Abhi hum dummy function pass karenge

📝 **CODE:**
```tsx
"use client";

import { useSearchParams } from "next/navigation";
import { useActionState } from "react";

export default function ResetPasswordPage() {
  const searchParams = useSearchParams();
  const token = searchParams.get("token");

  const [state, formAction] = useActionState(async () => {
    console.log("Reset password form submitted!");
    return {};
  }, null);

  if (!token) {
    return (
      <div className="max-w-md mx-auto mt-20 p-6 border rounded-lg">
        <h1 className="text-2xl font-bold mb-6">Reset Password</h1>
        <p className="text-red-500">Invalid or expired reset link</p>
      </div>
    );
  }

  return (
    <div className="max-w-md mx-auto mt-20 p-6 border rounded-lg">
      <h1 className="text-2xl font-bold mb-6">Reset Password</h1>

      <form action={formAction} className="space-y-4">
        <div>
          <label className="block mb-2">New Password</label>
          <input
            type="password"
            name="password"
            required
            className="w-full px-3 py-2 border rounded"
          />
        </div>

        <div>
          <label className="block mb-2">Confirm Password</label>
          <input
            type="password"
            name="confirmPassword"
            required
            className="w-full px-3 py-2 border rounded"
          />
        </div>

        <button
          type="submit"
          className="w-full bg-blue-500 text-white py-2 rounded"
        >
          Reset Password
        </button>

        {state?.error && (
          <p className="text-red-500">{state.error}</p>
        )}

        {state?.success && (
          <p className="text-green-500">{state.message}</p>
        )}
      </form>
    </div>
  );
}
```
