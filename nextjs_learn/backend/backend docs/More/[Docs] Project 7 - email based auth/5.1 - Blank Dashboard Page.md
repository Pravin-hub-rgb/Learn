## Step 1: Dashboard Page Banaye

✅ Ab `app` folder ke andar `(protected)` naam ka naya folder banaye.
Uske andar `dashboard` folder banaye.
Uske andar `page.tsx` file banaye.

✅ Structure hoga yeh:
```
app/
  (protected)/
    dashboard/
      page.tsx
```

> 🧠 Kyun `(protected)` group banaya hai?
> Yeh sirf organization ke liye hai. URL mein nahi aayega.
> `app/(protected)/dashboard/page.tsx` ka URL hoga sirf `/dashboard`
> Is group mein saare protected pages aayenge aage jaake.
