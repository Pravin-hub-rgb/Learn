# Doc 5.3 — JWT explicitly implement karo auth.ts mein

## Abhi Tak Kya Hua

5.2 mein samjha — JWT data cookie mein real hai — disk pe save hota hai — hawa mein nahi. Abhi `auth.ts` mein kuch explicitly nahi likha — sab Next-Auth ke defaults pe chal raha hai.

---

## Broad Goal — Yeh Doc Mein Kya Karenge

Explicitly likhenge `auth.ts` mein:
- `session.strategy: "jwt"` — clearly batao JWT use karna hai
- `maxAge` — session kitne time tak chalegi

---

## Pehle ek sawaal — explicitly likhne ki zaroorat kyun?

Abhi kaam toh kar raha hai bina kuch likhe. Toh kyun likhein?

Socho kal tune database add kar liya — adapter lagaya — Next-Auth ne automatically database session pe switch kar liya — tu chahta tha JWT — par switch ho gaya bina bataye.

Explicitly likhne se — **Next-Auth ko clearly pata hai tu kya chahta hai** — defaults pe depend nahi karna padega.

Aur `maxAge` — abhi Next-Auth ka default 30 din ka session hai — tune set nahi kiya — tujhe pata bhi nahi tha. Explicitly likhne se tu control mein hai.

---

## Step 1 — `session.strategy` explicitly likho

`auth.ts` kholo — abhi kuch aisa hai:

```ts
import NextAuth from "next-auth"
import GitHub from "next-auth/providers/github"

export const { handlers, signIn, signOut, auth } = NextAuth({
  providers: [GitHub],
  pages: {
    signIn: "/login",
  },
})
```

`session` config add karo:

```ts
import NextAuth from "next-auth"
import GitHub from "next-auth/providers/github"

export const { handlers, signIn, signOut, auth } = NextAuth({
  providers: [GitHub],
  session: {
    strategy: "jwt",
  },
  pages: {
    signIn: "/login",
  },
})
```

`session` — ek object hai — session ke baare mein config yahan likhte hain.

`strategy: "jwt"` — Next-Auth ko clearly bol diya — "JWT use karna hai — database nahi."

App test karo:

```
http://localhost:3000
```

Login karo — sab waise hi kaam kare jaise pehle? ✅ — Kuch change nahi hua — sirf explicitly likh diya.

---

## Step 2 — `maxAge` add karo

`maxAge` — session kitne seconds tak valid rahegi.

```ts
import NextAuth from "next-auth"
import GitHub from "next-auth/providers/github"

export const { handlers, signIn, signOut, auth } = NextAuth({
  providers: [GitHub],
  session: {
    strategy: "jwt",
    maxAge: 30 * 24 * 60 * 60,
  },
  pages: {
    signIn: "/login",
  },
})
```

`maxAge: 30 * 24 * 60 * 60` — yeh seconds mein hota hai. Calculate karo:

```
30        → 30 din
* 24      → har din 24 ghante
* 60      → har ghante 60 minute
* 60      → har minute 60 seconds
= 2592000 seconds = 30 din
```

Aise likhte hain taaki clearly pata chale — "30 din" — seedha `2592000` likhte toh samajh nahi aata.

App test karo — sab waise hi kaam kare? ✅

---

## Summary

- `session.strategy: "jwt"` — explicitly JWT use karne ke liye
- `maxAge` — session expiry seconds mein — `30 * 24 * 60 * 60` = 30 din
- Explicitly likhne se defaults pe depend nahi karna padta
