## Broader Goal — Aaj Hum Kya Karne Wale Hain
Hum chahte hain ki jab user “Login with GitHub” button dabaye, toh woh GitHub pe jaaye, wahan login kare, aur wapas hamari app pe aa jaye.  
Iske liye hum **Auth.js** library use karenge (jo yeh sab automatically handle karti hai).

**Kyun library use kar rahe hain?**  
Agar hum khud se banayenge toh bohot complicated ho jayega (redirect, callback, tokens wagairah).  
Library is problem ko solve karti hai — hum bas button laga denge.

Ab chhote steps mein karte hain.

### Step 1: Auth.js install karo
Terminal kholo (project folder mein) aur yeh command run karo:

```bash
npm install next-auth@beta
```

(ya agar bun use kar rahe ho to `bun add next-auth@beta`)

**Yeh kya kiya?**  
- `next-auth` library install ki  
- `@beta` isliye kyunki latest version abhi beta mein hai (lekin stable hai use karne ke liye)

**Try karo:**  
`package.json` file kholo aur dekho kya add hua hai.  
`"next-auth": "^5.0.0-beta.something"` aisa kuch dikhega.

### Step 2: Ek secret banao (security ke liye)
Auth.js ko ek secret chahiye taaki login safe rahe.

Terminal mein yeh run karo:

```bash
npx auth secret
```

Yeh ek lambi string dega (jaise 32-40 characters ki random cheez).  
Ab project mein `.env.local` file banao (agar nahi hai to bana lo) aur usme yeh paste kar do:

```env
AUTH_SECRET=woh-lambi-string-jo-terminal-me-ayi
```

**Yeh kyun zaroori hai?**  
Yeh secret login ke cookies aur tokens ko safe banata hai.  
Bina iske login kaam nahi karega.

**Try karo:**  
`.env.local` file save karo.  
Server restart karo (Ctrl+C kar ke phir `npm run dev` ya `bun dev`).

### Step 3: Sabse pehla button banao (Login with GitHub)
Ab `app/page.tsx` file kholo (jo home page hai).

Pehle sirf yeh code paste karo (purana content hata ke):

```tsx
'use client'   // ← yeh line important hai, isse page client side ban jayegi

export default function Home() {
  return (
    <div style={{ padding: '50px', textAlign: 'center' }}>
      <h1>Welcome to our app</h1>
      <button 
        onClick={() => alert("Button dabaya!")}
      >
        Login with GitHub
      </button>
    </div>
  )
}
```

**Yeh kya kiya?**
- `'use client'` likha → isse page client component ban gayi (kyunki button pe click karna hai)
- Ek simple button banaya jo abhi sirf alert dikhata hai

**Try karo:**  
Browser mein `http://localhost:3000` kholo.  
Button dikhega? Click karo → alert aata hai?  
Agar haan → next step ready.

### Step 4: Ab real login button banao (Auth.js ka signIn use karo)
Abhi hum alert ki jagah real login karayenge.

Pehle top pe yeh import add karo:

```tsx
import { signIn } from "next-auth/react"
```

Phir button change karo:

```tsx
<button 
  onClick={() => signIn("github")}
>
  Login with GitHub
</button>
```

**Yeh kya hua?**
- `signIn("github")` bol raha hai: “Bhai GitHub se login kara de”
- Auth.js automatically GitHub pe redirect kar dega

**Try karo abhi:**
- Save karo
- Browser refresh karo
- Button pe click karo

Kya hota hai?  
Tumhe GitHub login page pe le jana chahiye.  
Agar nahi gaya ya error aaya (jaise blank page ya 404) — console kholo (F12) aur mujhe batao kya likha aa raha hai.
