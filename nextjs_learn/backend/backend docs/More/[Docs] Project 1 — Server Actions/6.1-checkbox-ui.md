# Doc 6.1 — Checkbox Add Karo — Wahi `<li>` Upgrade Hogi

---

## Abhi Tak Kya Hua

Todos screen pe dikh rahe hain. Coder soochega — "Todo complete karna hai. Har todo ke saath checkbox hona chahiye."

---

## Step 1 — Checkbox Add Karo

`page.tsx` mein wahi `<li>` mein checkbox add karo:

```tsx
<li key={todo.id} className="flex items-center gap-2 p-3 border rounded">
  <input
    type="checkbox"
    defaultChecked={todo.done}
  />
  <span>{todo.title}</span>
</li>
```

`defaultChecked={todo.done}` — database mein jo value hai woh dikhao.

Save karo — screen pe har todo ke saath checkbox dikh raha hai ✅

---

## Step 2 — Click Karo — Kuch Nahi Hota

Checkbox click karo — visually check/uncheck hota hai.

Refresh karo — wapas pehle wali state. Database mein kuch nahi gaya.

---

## Step 3 — Action Chahiye — `form action` Try Karo

"`addTodo` mein `form action` se connect kiya tha — wahi yahan bhi karte hain."

`updateTodo` action `actions.ts` mein banao — abhi sirf console.log:

```ts
export async function updateTodo() {
  console.log('updateTodo chala!')
}
```

Checkbox ke aas paas `form` wrap karo:

```tsx
<li key={todo.id} className="flex items-center gap-2 p-3 border rounded">
  <form action={updateTodo}>
    <input
      type="checkbox"
      defaultChecked={todo.done}
    />
  </form>
  <span>{todo.title}</span>
</li>
```

`updateTodo` import karo — checkbox click karo — terminal mein dekho.

**Kuch nahi aaya.** Log nahi aaya.

---

## Kyun Nahi Aaya?

`form action` sirf tab submit hota hai jab:
- Submit button click ho
- Ya user Enter dabaaye

Checkbox click karna form submit nahi karta. Isliye `updateTodo` call hi nahi hua.

---

## Submit Button Add Karo — Try Karo

Temporarily ek submit button add karo — sirf test ke liye:

```tsx
<form action={updateTodo}>
  <input
    type="checkbox"
    defaultChecked={todo.done}
  />
  <button type="submit">✓</button>
</form>
```

Ab button click karo — terminal mein:

```
updateTodo chala!
```

Log aa gaya — matlab `form action` kaam karta hai — par **button click pe**, checkbox click pe nahi.

Yeh button UI mein nahi chahiye — awkward lagega. Button hata do.

---

## Toh Kya Karein?

Checkbox click hote hi — **turant** action chalna chahiye. Koi button nahi.

Iske liye `onChange` use karte hain — "jab bhi checkbox ka state badle — yeh function chalo."

Par `onChange` browser ki cheez hai — Server Component mein nahi hoti.

Toh ek alag **Client Component** banana padega — `TodoItem.tsx`.

Yahi hoga **Doc 6.2 mein.**

---

## Summary

- Checkbox add kiya — `defaultChecked` se database value reflect ki
- `form action` try kiya — submit button pe kaam kiya — checkbox click pe nahi
- `form action` yahan kaam nahi karega — checkbox submit trigger nahi karta
- Solution — `onChange` + Client Component — Doc 6.2 mein
