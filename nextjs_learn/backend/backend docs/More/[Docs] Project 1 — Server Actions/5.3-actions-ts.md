# Doc 5.3 — Actions Ko Organize Karo — `actions.ts`

---

## Abhi Tak Kya Hua

Todo add ho raha hai, dikh raha hai — sab kaam kar raha hai.

Par `TodoForm.tsx` abhi aisa hai:

```tsx
import { prisma } from '@/lib/prisma'
import { revalidatePath } from 'next/cache'

export default function TodoForm() {

  async function addTodo(formData: FormData) {
    "use server"
    const title = formData.get('title') as string
    await prisma.todo.create({ data: { title } })
    revalidatePath('/')
  }

  return (
    <form action={addTodo} className="flex gap-2">
      ...
    </form>
  )
}
```

Abhi ek function hai — theek hai. Par aage `updateTodo`, `deleteTodo` bhi aayenge — sab `TodoForm.tsx` mein rakhe toh file badi aur messy ho jaayegi.

Socho — `TodoForm.tsx` ka kaam kya hai? **UI dikhana.** Database ka kaam uska nahi.

Toh server actions alag file mein nikaal do — `actions.ts`.

## `"use server"` — Ab File Ke Top Pe

Abhi `TodoForm.tsx` mein `"use server"` **function ke andar** likhte the — ek function ke liye.

Jab alag file banate hain — `"use server"` **file ke top pe** likh sakte hain. Iska matlab — poori file ke saare functions server pe chalenge. Har function mein alag se likhne ki zaroorat nahi.

```ts
"use server"   // ← ek baar — poori file server-only

export async function addTodo() { ... }
export async function updateTodo() { ... }   // baad mein
export async function deleteTodo() { ... }   // baad mein
```

---

## `addTodo` Move Karo

`TodoForm.tsx` se `addTodo` cut karo — `actions.ts` mein paste karo:

```ts
"use server"

import { prisma } from '@/lib/prisma'
import { revalidatePath } from 'next/cache'

export async function addTodo(formData: FormData) {
  const title = formData.get('title') as string
  await prisma.todo.create({ data: { title } })
  revalidatePath('/')
}
```

`export` lagao — taaki `TodoForm.tsx` import kar sake.

`"use server"` andar se hata diya — file ke top pe hai — kaam same hoga.

---

## `TodoForm.tsx` Update Karo

`addTodo` hata diya — ab import karo:

```tsx
import { addTodo } from './actions'

export default function TodoForm() {
  return (
    <form action={addTodo} className="flex gap-2">
      <input
        type="text"
        name="title"
        placeholder="Kya karna hai?"
        className="border border-gray-300 rounded px-3 py-2 flex-1"
      />
      <button
        type="submit"
        className="bg-blue-500 text-white px-4 py-2 rounded hover:bg-blue-600"
      >
        Add Todo
      </button>
    </form>
  )
}
```

`TodoForm.tsx` ab sirf UI hai — clean.
