# Doc 7.1 — Delete Button Add Karo — Wahi `TodoItem` Upgrade Hogi

---

## Abhi Tak Kya Hua

`TodoItem.tsx` mein checkbox hai — todo complete kar sakte hain.

Coder soochega — "Todo delete karna hai. Delete button chahiye har todo pe."

---

## Delete Button Add Karo

`TodoItem.tsx` mein wahi `<li>` mein button add karo — title ke baad:

```tsx
<li className="flex items-center gap-2 p-3 border rounded">
  <input
    type="checkbox"
    defaultChecked={todo.done}
    disabled={isPending}
    onChange={(e) => {
      startTransition(() => {
        updateTodo(todo.id, e.target.checked)
      })
    }}
  />
  <span>{todo.title}</span>
  <button>Delete</button>
</li>
```

Save karo — screen pe har todo ke saath Delete button dikh raha hai ✅

---

## Button Ko Action Se Connect Karo

Checkbox ke liye `onChange` use kiya — `startTransition` bhi lagaya.

Par delete button alag hai — yeh ek proper button hai. Button click = form submit. Toh yahan `form action` kaam karega — `startTransition` ki zaroorat nahi.

Button ko `form` se wrap karo:

```tsx
<form action={deleteTodo}>
  <button type="submit">Delete</button>
</form>
```

Par `deleteTodo` ko `id` kaise milega? Hidden input se — wahi tarika jo update mein samjha tha:

```tsx
<form action={deleteTodo}>
  <input type="hidden" name="id" value={todo.id} />
  <button type="submit">Delete</button>
</form>
```

Poora `<li>` ab:

```tsx
<li className="flex items-center gap-2 p-3 border rounded">
  <input
    type="checkbox"
    defaultChecked={todo.done}
    disabled={isPending}
    onChange={(e) => {
      startTransition(() => {
        updateTodo(todo.id, e.target.checked)
      })
    }}
  />
  <span>{todo.title}</span>
  <form action={deleteTodo}>
    <input type="hidden" name="id" value={todo.id} />
    <button type="submit">Delete</button>
  </form>
</li>
```

---

## Summary

- Wahi `TodoItem` mein delete button add kiya
- Button ke liye `form action` — `startTransition` ki zaroorat nahi
- Hidden input se `id` pass kiya
- `deleteTodo` action abhi bani nahi — Doc 7.2 mein banayenge

## Agla Step

UI ready hai. Ab `deleteTodo` action banani hai.
