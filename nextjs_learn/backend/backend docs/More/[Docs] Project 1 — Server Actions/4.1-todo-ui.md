# Doc 4.1 — Sirf UI — Input Aur Button

---

## Abhi Tak Kya Hua

Setup poora ho gaya — Next.js, Neon, Prisma, database table sab ready hai.

Ab asli kaam shuru hota hai — **Todo Add karna.**

---

## Pehle Kya Karein?

Coder ki tarah socho — "Mujhe todo add karna hai. Toh sabse pehle kya chahiye?"

User ko kuch type karna hai — toh **input** chahiye.
User ko submit karna hai — toh **button** chahiye.

Bas. Abhi sirf itna. Database, logic, kuch nahi — pehle screen pe dikhao.

---

## Kahan Likhein Yeh UI?

`page.tsx` mein seedha likh sakte hain — par yeh todo form ek apni cheez hai. Alag component banao taaki clean rahe.

`app/` ke andar ek naya file banao — `TodoForm.tsx`:

```
app/
├── TodoForm.tsx    ← naya
└── page.tsx
```

---

## `TodoForm.tsx` Banao

Pehle sirf ek basic component — koi logic nahi:

```tsx
export default function TodoForm() {
  return (
    <div>
      
    </div>
  )
}
```

---

## Input Add Karo

User yahan type karega — `input` element:

```tsx
export default function TodoForm() {
  return (
    <div>
      <input
        type="text"
        placeholder="Kya karna hai?"
      />
    </div>
  )
}
```

---

## Button Add Karo

Submit karne ke liye:

```tsx
export default function TodoForm() {
  return (
    <div>
      <input
        type="text"
        placeholder="Kya karna hai?"
      />
      <button type="submit">
        Add Todo
      </button>
    </div>
  )
}
```

---

## Tailwind Se Thoda Theek Karo

Abhi dono elements chipke hue hain — thoda spacing aur styling do:

```tsx
export default function TodoForm() {
  return (
    <div className="flex gap-2">
      <input
        type="text"
        placeholder="Kya karna hai?"
        className="border border-gray-300 rounded px-3 py-2 flex-1"
      />
      <button
        type="submit"
        className="bg-blue-500 text-white px-4 py-2 rounded hover:bg-blue-600"
      >
        Add Todo
      </button>
    </div>
  )
}
```

`flex gap-2` — dono side by side, thoda gap.
`flex-1` — input baaki saari jagah le le.
`hover:bg-blue-600` — button pe hover pe thoda dark blue.

---

## `page.tsx` Mein Import Karo

Ab yeh form screen pe dikhana hai — `page.tsx` mein import karo:

```tsx
import TodoForm from './TodoForm'

export default function Home() {
  return (
    <main className="max-w-xl mx-auto mt-10 p-4">
      <h1 className="text-2xl font-bold mb-6">My Todos</h1>
      <TodoForm />
    </main>
  )
}
```

`max-w-xl mx-auto` — center mein, zyada wide nahi.
`mt-10` — upar se thoda space.
