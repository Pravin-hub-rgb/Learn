# Doc 8.2 — Error Handle Karo — Kuch Galat Ho Gaya

---

## Abhi Tak Kya Hua

Loading state aa gayi — user ko pata chalta hai kaam chal raha hai.

Par ek aur case hai — kya hoga agar user khali input submit kare? Ya database down ho jaaye?

---

## Practically Dekho — Khali Input Submit Karo

`TodoForm.tsx` mein input khali chhod ke button dabao.

Kya hua? **Khali todo add ho gaya** — list mein ek blank item aa gaya.

Yeh nahi hona chahiye. Khali title database mein nahi jaana chahiye.

---

## Pehle Validation — `addTodo` Mein Check Lagao

`actions.ts` mein `addTodo` mein title check karo — khali hai toh error return karo:

```ts
export async function addTodo(prevState: unknown, formData: FormData) {
  const title = formData.get('title') as string

  if (!title || title.trim() === '') {
    return { error: 'Title khali nahi ho sakta.' }
  }

  await prisma.todo.create({ data: { title } })
  revalidatePath('/')
  return null
}
```

`title.trim() === ''` — sirf spaces bhi khali count honge — `'   '` bhi invalid hai.

Ab khali submit karo — database mein nahi gaya. Par **screen pe kuch nahi dikhega** — error return kiya par abhi kahan gaya pata nahi.

---

## `state` Mein Kya Aaya — Pehle Dekho

Yaad hai `useActionState` se `state` milta tha? Jab `addTodo` kuch return karta hai — woh `state` mein aa jaata hai.

`TodoForm.tsx` mein `console.log` lagao — dekho `state` mein kya hai:

```tsx
export default function TodoForm() {
  const [state, formAction, isPending] = useActionState(addTodo, null)

  console.log('state:', state)   // ← yeh add karo

  return (
    ...
  )
}
```

Ab khali input submit karo — **browser console** (F12) mein dekho:

```
state: { error: "Title khali nahi ho sakta." }
```

`state` mein woh object aa gaya jo `addTodo` ne return kiya — `error` property ke saath.

Toh `state.error` mein message hai — isko UI mein dikhao.

Console.log hata do — ab kaam aaya nahi chahiye.

---

## `state.error` UI Mein Dikhao

Form ke neeche error dikhao:

```tsx
return (
  <div>
    <form action={formAction} className="flex gap-2">
      <input
        type="text"
        name="title"
        placeholder="Kya karna hai?"
        className="border border-gray-300 rounded px-3 py-2 flex-1"
      />
      <button
        type="submit"
        disabled={isPending}
        className="bg-blue-500 text-white px-4 py-2 rounded hover:bg-blue-600 disabled:opacity-50"
      >
        {isPending ? 'Adding...' : 'Add Todo'}
      </button>
    </form>
    {state?.error && (
      <p className="text-red-500 text-sm mt-2">{state.error}</p>
    )}
  </div>
)
```

`state?.error` — `state` shuru mein `null` hota hai — `?.` se safely check karo — hai toh dikhao, nahi hai toh kuch nahi.

`text-red-500` — red text — error ke liye.

Ab khali submit karo — **"Title khali nahi ho sakta."** screen pe dikh raha hai ✅

## `try/catch` — Database Error Bhi Handle Karo

Validation toh ho gayi. Par ek aur case — database down ho jaaye? Network issue ho?

Abhi error aayegi — app crash karega — user ko kuch nahi dikhega.

`try/catch` lagao — database operation wrap karo:

```ts
export async function addTodo(prevState: unknown, formData: FormData) {
  const title = formData.get('title') as string

  if (!title || title.trim() === '') {
    return { error: 'Title khali nahi ho sakta.' }
  }

  try {
    await prisma.todo.create({ data: { title } })
    revalidatePath('/')
    return null
  } catch (error) {
    return { error: 'Todo add nahi hua — dobara try karo.' }
  }
}
```

`try` — yeh kaam karo.

`catch` — agar kuch bhi galat hua — yahan pakad lo — error **throw mat karo**, **return karo** — taaki `state` mein aaye aur UI mein dikh sake.

**Khali input submit karo** — "Title khali nahi ho sakta." dikhega ✅

**Kuch type karo** — successfully add ho gaya — error nahi dikhega ✅

## Poora `actions.ts`

```ts
"use server"

import { prisma } from '@/lib/prisma'
import { revalidatePath } from 'next/cache'

export async function addTodo(prevState: unknown, formData: FormData) {
  const title = formData.get('title') as string

  if (!title || title.trim() === '') {
    return { error: 'Title khali nahi ho sakta.' }
  }

  try {
    await prisma.todo.create({ data: { title } })
    revalidatePath('/')
    return null
  } catch (error) {
    return { error: 'Todo add nahi hua — dobara try karo.' }
  }
}

export async function updateTodo(id: number, done: boolean) {
  await prisma.todo.update({ where: { id }, data: { done } })
  revalidatePath('/')
}

export async function deleteTodo(formData: FormData) {
  const id = Number(formData.get('id'))
  await prisma.todo.delete({ where: { id } })
  revalidatePath('/')
}
```

## Summary

- Khali input submit kiya — todo add ho gaya — problem thi
- Validation lagayi — `title.trim() === ''` — khali title block kiya
- `console.log('state:', state)` — dekha ki `state` mein `{ error: "..." }` aaya
- `state.error` — UI mein dikhaya — `?.` se safely check kiya
- `try/catch` — database error bhi handle kiya — throw nahi, return kiya
