# Doc 4.4 — Database Mein Save Karo

---

## Abhi Tak Kya Hua

`TodoForm.tsx` mein `form action` + `FormData` ban gaya — `addTodo` server pe chal raha hai, `formData.get('title')` se value terminal mein aa rahi hai.

Abhi sirf `console.log` hai. Ab database mein save karte hain.

---

## Database Se Baat Karne Ke Liye Kya Chahiye?

"Database mein save karna hai — toh `prisma` chahiye."

`lib/prisma.ts` mein banaya tha — wahan se import karo:

```tsx
import { prisma } from '@/lib/prisma'

export default function TodoForm() {

  async function addTodo(formData: FormData) {
    "use server"
    const title = formData.get('title')
    console.log('Title:', title)
  }

  return (
    <form action={addTodo} className="flex gap-2">
      <input
        type="text"
        name="title"
        placeholder="Kya karna hai?"
        className="border border-gray-300 rounded px-3 py-2 flex-1"
      />
      <button
        type="submit"
        className="bg-blue-500 text-white px-4 py-2 rounded hover:bg-blue-600"
      >
        Add Todo
      </button>
    </form>
  )
}
```

---

## `prisma.todo.create()` — Naya Todo Banao

`console.log` ki jagah Prisma se naya todo banao:

```tsx
async function addTodo(formData: FormData) {
  "use server"
  const title = formData.get('title')

  await prisma.todo.create({
    data: {
      title: title
    }
  })
}
```

Yeh tod ke samjho —

`prisma.todo` — `todos` table pe kaam karna hai.

`.create()` — naya row daalo.

`data: {}` — is object mein batao naye row mein kya hoga. Sirf `title` de rahe hain — `id` aur `done` schema mein `@default` se automatic hain.

`await` — database mein save hone mein time lagta hai — wait karo.

---

## TypeScript Warning — Fix Karo

`title` pe red squiggly line aayegi:

```
Type 'FormDataEntryValue | null' is not assignable to type 'string'
```

Kyun? `formData.get('title')` do cheezein return kar sakta hai:
- Value mili — `string`
- Input khali tha — `null`

Par `prisma.todo.create()` mein `title` sirf `string` chahiye — `null` nahi chalega.

Fix — `as string` lagao:

```tsx
const title = formData.get('title') as string
```

`as string` — TypeScript ko bolta hai "main guarantee deta hoon yeh string hai."

---

## Shorthand

JavaScript mein — jab variable ka naam aur object key same ho:

```tsx
data: {
  title: title  // dono same hain
}
```

Shorthand mein sirf ek baar likho:

```tsx
data: {
  title  // same cheez
}
```

Final `addTodo`:

```tsx
async function addTodo(formData: FormData) {
  "use server"
  const title = formData.get('title') as string

  await prisma.todo.create({
    data: { title }
  })
}
```
