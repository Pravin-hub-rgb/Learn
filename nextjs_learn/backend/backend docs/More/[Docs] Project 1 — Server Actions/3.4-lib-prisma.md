# Doc 3.4 — `lib/prisma.ts` Banao

---

## Abhi Tak Kya Hua

Prisma install ho gaya. `schema.prisma` aur `prisma.config.ts` ban gayi. `.env` mein connection string hai.

Kai jagah database use hoga app mein — todo add karna, fetch karna, delete karna. Har jagah Prisma chahiye. Toh ek jagah connection banao aur reuse karo — `lib/prisma.ts`.

---

## Step 1 — Pehle Sirf Connection Banao

`lib/prisma.ts` file banao. Pehle sirf itna likho:

```ts
export const prisma = new PrismaClient()
```

---

## Step 2 — `PrismaClient` Kahan Se Import Karein?

Abhi TypeScript error dega — `PrismaClient` defined nahi hai.

Yaad hai `npx prisma generate` chalane se `app/generated/prisma/` folder bana tha? `PrismaClient` wahan se aata hai:

```ts
import { PrismaClient } from '../app/generated/prisma/client'

export const prisma = new PrismaClient()
```

---

## Step 3 — Adapter Bhi Chahiye

Dev server chalao — yeh error aayega:

```
PrismaClient needs to be constructed with a valid PrismaClientOptions
```

Prisma v7 mein sirf `new PrismaClient()` nahi chalega. Usse batana padta hai — database se kaise connect kare.

Iske liye `PrismaPg` adapter use karte hain. Par pehle connection string ek variable mein rakho — cleaner lagta hai:

```ts
import { PrismaClient } from '../app/generated/prisma/client'
import { PrismaPg } from '@prisma/adapter-pg'

const connectionString = process.env.DATABASE_URL!
```

`!` — yeh TypeScript ko bolta hai "main guarantee deta hoon yeh `undefined` nahi hoga." `.env` mein `DATABASE_URL` hai — toh safe hai.

Ab adapter banao aur pass karo:

```ts
import { PrismaClient } from '../app/generated/prisma/client'
import { PrismaPg } from '@prisma/adapter-pg'

const connectionString = process.env.DATABASE_URL!

const adapter = new PrismaPg({ connectionString })

export const prisma = new PrismaClient({ adapter })
```

Bas — yahi final file hai. Simple, clean, official docs wala pattern.

---

## Step 4 — `prisma generate` Chalao

```bash
npx prisma generate
```

Output:

```
✔ Generated Prisma Client (v7.5.0) to ./app/generated/prisma
```

---

## Step 5 — Verify Karo

`app/page.tsx` mein temporarily yeh likho:

```ts
import { prisma } from '@/lib/prisma'

export default async function Home() {
  console.log('Prisma Connected:', prisma)
  return <div>Todo App</div>
}
```

`npm run dev` chalao. Terminal mein dekho:

```
Prisma Connected: PrismaClient { ... }
GET / 200
```

✅ Connection kaam kar raha hai — yeh lines ab hata do.

---

## Summary

- `lib/prisma.ts` banai — simple, official pattern
- `PrismaClient` `app/generated/prisma/client` se import kiya — Prisma v7 mein yahan hota hai
- `connectionString` alag variable mein rakha — cleaner hai
- `PrismaPg` adapter pass kiya — Prisma v7 mein zaroori hai
- Verify kiya — kaam kar raha hai

---

## Agla Step

Yeh pattern simple hai — par development mein ek problem aa sakti hai. Woh kya hai aur practically kaise hoti hai — yahi dekhenge **Doc 3.5 mein.**
