## `request.nextUrl` kya hai?

Proxy mein `request` parameter milta hai — `NextRequest` type ka. Iske andar ek property hai — `nextUrl` — yeh poora URL object hai.

Pehle `console.log` karke dekho kya milta hai:

```ts
import { NextRequest, NextResponse } from "next/server"

export function proxy(request: NextRequest) {
  console.log(request.nextUrl)
  return NextResponse.next()
}

export const config = {
  matcher: ["/((?!_next/static|_next/image|favicon.ico).*)"],
}
```

Browser mein visit karo:

```
http://localhost:3000/dashboard?tab=settings&user=123
```

Terminal mein kuch aisa dikhega:

```js
URL {
  href: 'http://localhost:3000/dashboard?tab=settings&user=123',
  pathname: '/dashboard',
  search: '?tab=settings&user=123',
  searchParams: URLSearchParams { 'tab' => 'settings', 'user' => '123' },
  ...
}
```

Bahut saari cheezein hain — aaj `pathname` aur `searchParams` dekhenge.

## `pathname` — URL ka path part

`pathname` — URL mein domain ke baad wala part — `?` se pehle:

```
http://localhost:3000/dashboard?tab=settings
                      ↑
                  pathname = "/dashboard"
```

Directly nikal sakte ho:

```ts
const pathname = request.nextUrl.pathname
console.log(pathname) // → "/dashboard"
```

**Practical use** — yeh check karna ki user kaunse page pe ja raha hai:

```ts
import { NextRequest, NextResponse } from "next/server"

export function proxy(request: NextRequest) {
  const pathname = request.nextUrl.pathname
  console.log("User ja raha hai:", pathname)
  return NextResponse.next()
}

export const config = {
  matcher: ["/((?!_next/static|_next/image|favicon.ico).*)"],
}
```

Test karo alag alag URLs pe:

```
http://localhost:3000/          → "User ja raha hai: /"
http://localhost:3000/dashboard → "User ja raha hai: /dashboard"
http://localhost:3000/profile   → "User ja raha hai: /profile"
```

## `searchParams` — URL ke query parameters

`searchParams` — URL mein `?` ke baad wala part:

```
http://localhost:3000/products?category=shoes&color=red
                               ↑
                       searchParams mein hai yeh sab
```

`searchParams.get()` se koi bhi parameter nikalo:

```ts
import { NextRequest, NextResponse } from "next/server"

export function proxy(request: NextRequest) {
  const pathname = request.nextUrl.pathname
  const category = request.nextUrl.searchParams.get("category")
  console.log("pathname:", pathname)
  console.log("category:", category)
  return NextResponse.next()
}

export const config = {
  matcher: ["/((?!_next/static|_next/image|favicon.ico).*)"],
}
```

`.get("category")` — `get` method hai — string mein parameter ka naam do — value milti hai. Agar woh parameter URL mein hai hi nahi — `null` milta hai.

Test karo:

```
http://localhost:3000/products?category=shoes
```

Terminal mein:

```
pathname: /products
category: shoes
```

Ab bina `category` ke:

```
http://localhost:3000/products
```

Terminal mein:

```
pathname: /products
category: null
```

`null` — parameter tha hi nahi URL mein. ✅

## Practical example — condition URL se

Ab combine karo — `pathname` se check karo user kahan ja raha hai:

```ts
import { NextRequest, NextResponse } from "next/server"

export function proxy(request: NextRequest) {
  const pathname = request.nextUrl.pathname

  if (pathname.startsWith("/dashboard")) {
    console.log("Dashboard route pe hai user")
  }

  return NextResponse.next()
}

export const config = {
  matcher: ["/((?!_next/static|_next/image|favicon.ico).*)"],
}
```

`.startsWith("/dashboard")` — string ka method hai — "kya yeh string `/dashboard` se shuru hoti hai?" — `true` ya `false` return karta hai.

Test karo:

```
http://localhost:3000/dashboard          → "Dashboard route pe hai user" ✅
http://localhost:3000/dashboard/settings → "Dashboard route pe hai user" ✅
http://localhost:3000/about              → kuch nahi ✅
```

---

## Summary

- `request.nextUrl` — poora URL object — isme sab kuch hai
- `request.nextUrl.pathname` — URL ka path — `/dashboard`, `/about`, etc.
- `request.nextUrl.searchParams.get("key")` — query parameter ki value — nahi hai toh `null`
- `.startsWith()` — check karo path kis cheez se shuru hota hai
