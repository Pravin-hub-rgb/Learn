## `NextResponse` kya hai

Abhi tak `NextResponse.next()` use kiya hai — matlab "kuch mat karo — request aage jaane do."

Par `NextResponse` mein aur bhi methods hain — teen main kaam kar sakta hai:

```
NextResponse.next()      → kuch mat karo — aage jaane do
NextResponse.redirect()  → user ko kisi aur URL pe bhejo
NextResponse.rewrite()   → URL same dikhao — content alag do
```

Aur response pe:
```
response.headers.set()   → response mein header add karo
response.cookies.set()   → response mein cookie set karo
```

Ek ek karke dekhenge — pehle briefly samjho dono main methods ka fark.

---

## `next()` vs `redirect()` vs `rewrite()` — fark kya hai?

Story se samjho — shopping mall ka security guard:

**`next()`** — "Theek hai — andar jaao":

```
User aaya → Guard ne check kiya → "Theek hai, jaao" → User mall mein gaya
URL browser mein: /dashboard  ✅
Content: /dashboard ka content
```

**`redirect()`** — "Nahi — woh wali building mein jaao":

```
User aaya → Guard ne check kiya → "Nahi, woh gate se jaao" → User wahan gaya
URL browser mein: /login  ← URL BADAL GAYA
Content: /login ka content
```

**`rewrite()`** — "Yahan ruko — main woh wali jagah ka content laata hoon":

```
User aaya → Guard ne check kiya → User wahan hi raha
URL browser mein: /dashboard  ← URL SAME RAHA
Content: /home ka content  ← CONTENT ALAG AAYA
```

`rewrite()` — user ko pata nahi chalta ki actually kaunsa page dikh raha hai.

---

## Kab kaunsa use karein

```
User login nahi hai — protected route pe gaya
→ redirect() — /login pe bhejo — URL bhi badlo

A/B testing — /home visit kiya
→ rewrite() — kuch users ko /home-v2 ka content do — URL same rakho

Sab theek hai — aage jaane do
→ next()
```

---

## `NextResponse` kahan se aata hai

Abhi tak import kar rahe hain — par ek baar clear karo:

```ts
import { NextResponse } from "next/server"
```

`"next/server"` — Next.js ka server-side package — `NextRequest` aur `NextResponse` dono yahan se aate hain. Yeh automatically available hai Next.js project mein — alag se install nahi karna.

---

## Summary

- `NextResponse.next()` — aage jaane do — kuch change nahi
- `NextResponse.redirect()` — user ko kisi aur URL pe bhejo — URL badalta hai browser mein
- `NextResponse.rewrite()` — URL same rakho — content alag do
- `response.headers.set()` — response mein header add karo
- `"next/server"` se import hota hai
